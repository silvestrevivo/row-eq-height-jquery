Bootstrap equal-height columns

This is a demo about a jQuery-based equal-height columns with Bootstrap's grid system.

Normal row(no equal-height)

Here we don't use .row-eq-height.

.row > .col-sm-4
.row > .col-sm-4
this is
a much
taller
column
than the others
.row > .col-sm-4

Row with equal-height columns

Here we use the custom .row-eq-height, defined with jQuery.

All of the columns will stretch vertically to occupy the same height as the tallest column.

.row > .col-sm-4
.row > .col-sm-4
this is
a much
taller
column
than the others
.row > .col-sm-4

Warning: Browser compatibility

The .row-eq-height doesn't use FlexBox, therefore is compatible with Internet Explorer.

Warning: Changes column wrapping behavior

This jQuery code is optimalized for width > 768px. Feel free to resize to see effect.
